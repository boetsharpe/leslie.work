# Leslie Work - Coming Soon Page

Simple "coming soon" page for www.leslie.work

## Deployment Options

### GitHub Pages
1. Create a GitHub repository
2. Push this code to the repository
3. Go to Settings â†’ Pages
4. Select your branch and save
5. Point your domain (www.leslie.work) to the GitHub Pages URL

### Netlify
1. Drag and drop the `index.html` file to [Netlify Drop](https://app.netlify.com/drop)
2. Or connect your GitHub repository
3. Add custom domain (www.leslie.work) in domain settings

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory
3. Add custom domain (www.leslie.work) in project settings
